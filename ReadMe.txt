Результат прошу отправить в ответном письме. 

Задание:
Используя среду разработки для Delphi не менее 7ой, реализовать
программу копирования файлов.
Основной функционал, который должен присутствовать(нюансы оставляем на
ваш выбор):
- Диалог выбора исходных файлов и места копирования.
- Прогресс копирования отображать в модальном окне, с двумя прогресс
барами - кол-во файлов и прогресс по каждому из файлов.
- В окне копирования отображать имя файла, его размер и общее время
копирования.
- Должна быть возможность прервать копирование, для этого использовать
потоки/нити.
- При успешном копировании файла заносить его имя, размер, время
копирования в таблицу базы данных Firebird 2.5
- После завершения всего копирования, результаты, занесенные в таблицу,
отображать на форме в табличке.

Разработку вести в локальном репозитории git или в облаке(GitHub и т.п.).
Решение высылать в виде zip-архива клона проекта или в виде ссылки на
публичный репозиторий.
